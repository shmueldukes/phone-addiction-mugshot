## Review Notes for App Store Review Team

**Important Testing Notes:**

1. **Screen Time Permission**: The app requires Screen Time API access. Please grant this permission when prompted to fully test the app.

2. **Camera Permission**: Front-facing camera access is required for the mugshot feature. Please grant camera permission.

3. **Test Flow:**
   - Grant Screen Time permission on first launch
   - Tap "Take Mugshot Now" on the home screen
   - Grant camera permission when prompted
   - Capture a test selfie
   - View the gallery to see saved mugshots
   - Test sharing feature from gallery detail view

4. **Deep Linking**: The app supports the URL scheme `phoneaddictionmugshot://` for shared content deep linking.

5. **Analytics**: The app uses anonymized analytics (PostHog/Mixpanel) - no personal data is transmitted.

6. **Contact Support**: Available in Settings > Contact Support

**Privacy Compliance:**
- All photos stored locally only
- No photos uploaded to servers
